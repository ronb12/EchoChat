# EchoChat Icons

This directory contains the app icons for EchoChat. You need to create the following icon files:

## Required Icons

- `icon-16x16.png` - 16x16 pixels
- `icon-32x32.png` - 32x32 pixels  
- `icon-72x72.png` - 72x72 pixels
- `icon-96x96.png` - 96x96 pixels
- `icon-128x128.png` - 128x128 pixels
- `icon-144x144.png` - 144x144 pixels
- `icon-152x152.png` - 152x152 pixels
- `icon-192x192.png` - 192x192 pixels
- `icon-384x384.png` - 384x384 pixels
- `icon-512x512.png` - 512x512 pixels
- `default-avatar.png` - Default user avatar

- Secondary: #1a1a2e (darker blue)
- Accent: #e94560 (red/pink)
- Background: #0f0f23 (very dark blue)

- **Figma**: Free design tool
- **Canva**: Easy icon creation
- **Adobe Illustrator**: Professional design
- **GIMP**: Free alternative to Photoshop
- **Online Icon Generators**: Various free tools

## Quick Setup

1. Create a 512x512 master icon
2. Use an icon generator to create all sizes
3. Place all files in this directory
4. Update manifest.json if needed

## Default Avatar

Create a simple default avatar for users without profile pictures:
- 48x48 pixels minimum
- Simple design (user silhouette or initials)
- Neutral colors
- PNG format with transparency
